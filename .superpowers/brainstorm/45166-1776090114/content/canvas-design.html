A

Full Excalidraw Experience

Embed the full @excalidraw/excalidraw component with all tools (shapes, arrows, text, freehand draw, sticky notes, images). Canvas fills the entire content area. Excalidraw's own toolbar replaces our Tiptap toolbar. Auto-save scene JSON to Document.content.

B

Excalidraw with Custom Toolbar

Same full Excalidraw but hide its default UI and build our own toolbar that matches the workspace header style. More polished integration but more work.

C

Excalidraw Default + Minimal Customization

Use Excalidraw as-is with default toolbar and UI. Just customize the theme colors to match our app. Fastest to implement, Excalidraw's UX is already excellent.
