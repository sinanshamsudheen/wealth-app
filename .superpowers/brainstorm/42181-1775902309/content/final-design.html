Owner
Full control — read, write, delete, configure, export, manage users within module
Manager
Manage work — read, write, assign, review, approve, export (no critical deletes)

Execute — read, create own, edit own, submit for review (limited scope)
Admin module access = Org Admin. Only users with a role in the Administration module can see the Administration sidebar item and access company settings, branding, and user management.

5. Complete Scope Summary

What we're building — at a glance
✅ In Scope (v1)
  • Contextual sidebar — morphs per module, "Your Daily" pinned at bottom
  • Company Profile — name, number, URL, currency, timezone, address, status, support email
  • Branding — small/large logo, header logo choice, brand color, email footer
  • Roles & Access — user list, invite flow, per-module role assignment (Owner/Manager/Analyst)
  • My Account — personal profile, security settings (password, 2FA, username), appearance
  • 3 system-defined roles — Owner, Manager, Analyst (no custom roles)
  • Admin = module — Administration access via Admin module Owner/Manager role
  • User invite via email — 3-step flow (details → roles → send)
⏳ Deferred (v2+)
  • Groups (bulk role assignment)
  • Custom roles
  • Audit log viewer
  • Activity log / login history
  • SSO / SAML integration
  • IP whitelisting
  • API key management
  • Data export / GDPR tools
  • Notification preferences (per module)
