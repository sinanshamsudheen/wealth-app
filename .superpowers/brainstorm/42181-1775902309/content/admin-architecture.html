RBAC Model Summary

How permissions work
Owner
  • Full read/write/delete
  • Configure module settings
  • Export & share data
  • Manage module users
  • View all records
Manager
  • Read/write (no delete critical)
  • Assign work to analysts
  • Approve/review submissions
  • View team records
  • Export data
Analyst
  • Read + create own records
  • Edit own work only
  • Submit for review
  • View assigned records
  • Limited exports
Note: Administration module itself is only accessible to users who have the Owner role in at least one module (i.e., they are an org admin). The "Org Admin" is not a separate role — it's derived from having Owner-level access.
